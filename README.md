# Anisubv2 🎬

**Anisubv2** হলো একটি উন্নত অটোমেটেড ভিডিও প্রসেসিং টুল যা ভিডিও থেকে সাবটাইটেল এক্সট্রাক্ট করা, সেগুলোকে বাংলায় অনুবাদ করা এবং ভিডিওর সাথে হার্ডকোড করে পুনরায় রেন্ডার করার সুবিধা প্রদান করে। এটি বিশেষভাবে অ্যানিমে বা বিদেশি ভাষার ভিডিওর জন্য তৈরি করা হয়েছে।

## ✨ মূল বৈশিষ্ট্যসমূহ (Key Features)

-   **অটোমেটেড সাবটাইটেল এক্সট্রাকশন:** ভিডিও সোর্স থেকে সরাসরি সাবটাইটেল খুঁজে বের করা।
-   **AI অনুবাদ:** Gemini, Grok বা OpenAI ব্যবহার করে সাবটাইটেল নিখুঁতভাবে বাংলায় অনুবাদ করা।
-   **হার্ডকোড সাবটাইটেল:** বিভিন্ন বাংলা ফন্ট (যেমন: Noto Sans Bengali, SolaimanLipi) ব্যবহার করে ভিডিওতে সাবটাইটেল যুক্ত করা।
-   **মাল্টি-প্ল্যাটফর্ম আপলোড:** প্রসেস করা ভিডিও সরাসরি **Telegram** এবং **Facebook**-এ আপলোড করার সুবিধা।
-   **মোবাইল ফ্রেন্ডলি:** এটি সম্পূর্ণ রেসপনসিভ এবং মোবাইল ব্রাউজার থেকে সহজেই ব্যবহার করা যায়।
-   **রিয়েল-টাইম প্রগ্রেস:** ভিডিও রেন্ডারিং এবং আপলোডের লাইভ প্রগ্রেস দেখার সুবিধা।

## 🛠 প্রযুক্তিগত কাঠামো (Tech Stack)

-   **Backend:** Python (Flask)
-   **Processing:** FFmpeg (ভিডিও রেন্ডারিংয়ের জন্য)
-   **Downloaders:** yt-dlp
-   **Frontend:** HTML5, CSS3, JavaScript
-   **Hosting:** Railway, Docker

## 📂 ফাইল স্ট্রাকচার (File Structure)

| ফাইল | বর্ণনা |
| :--- | :--- |
| `app.py` | মূল ফ্ল্যাস্ক অ্যাপ্লিকেশন এবং এন্ডপয়েন্টসমূহ। |
| `extractor.py` | ভিডিও সোর্স থেকে লিঙ্ক এবং সাবটাইটেল এক্সট্রাক্ট করার লজিক। |
| `translator.py` | AI ব্যবহার করে সাবটাইটেল অনুবাদের স্ক্রিপ্ট। |
| `uploader.py` | টেলিগ্রাম এবং ফেসবুকে ভিডিও আপলোড করার মডিউল। |
| `Dockerfile` | ডকার কন্টেইনার তৈরির কনফিগারেশন। |
| `railway.json` | Railway-তে হোস্ট করার জন্য কনফিগারেশন ফাইল। |

## 🚀 Railway-তে হোস্ট করার নিয়ম (Hosting on Railway)

Anisubv2 খুব সহজেই Railway-তে হোস্ট করা যায়। নিচের ধাপগুলো অনুসরণ করুন:

1.  **GitHub-এ কানেক্ট করুন:** আপনার Railway অ্যাকাউন্টে লগইন করে "New Project" এ ক্লিক করুন এবং এই রিপোজিটরিটির সাথে কানেক্ট করুন।
2.  **Environment Variables সেট করুন:** Railway-এর 'Variables' ট্যাবে নিচের কি-গুলো যুক্ত করুন:
    -   `GEMINI_API_KEY` (অনুবাদের জন্য)
    -   `TG_BOT_TOKEN` (টেলিগ্রাম আপলোডের জন্য)
    -   `FB_PAGE_TOKEN` (ফেসবুক আপলোডের জন্য)
3.  **Deploy:** Railway অটোমেটিক `Dockerfile` এবং `railway.json` ব্যবহার করে আপনার অ্যাপটি বিল্ড এবং রান করবে।

## 📱 মোবাইলে ব্যবহার (Mobile Usage)

এই টুলটি মোবাইলে ব্যবহারের জন্য বিশেষভাবে অপ্টিমাইজ করা হয়েছে।
-   যেকোনো মোবাইল ব্রাউজার (Chrome/Safari) দিয়ে আপনার হোস্ট করা লিঙ্কে প্রবেশ করুন।
-   ভিডিও লিঙ্ক দিন এবং 'Start' বাটনে ক্লিক করুন।
-   ব্যাকগ্রাউন্ডে প্রসেসিং চলবে এবং কাজ শেষ হলে আপনাকে টেলিগ্রাম বা ফেসবুকে লিঙ্ক প্রদান করবে।
